# P2P Compute: Peer Application

React, Electron, Chakra Frontend with a Flask Backend

## Installation

```sh
python3 -m venv venv
pip3 install -r requirements.txt

npm install
```

## Run

### Backend

```
. venv/bin/activate
cd backend 
python index.py
```

### Frontend

```
. venv/bin/activate
npm start
```
